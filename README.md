# CashMoov Financial Analytics: WalletOwner Dashboard Report

##  Project Overview
This business intelligence project was developed during my Data Analyst Internship at **CashMoov**. The core objective was to design, build, and deploy an end-to-end analytics solution to monitor, analyze, and reconcile financial transactions associated with Wallet Owners. 

The resulting Power BI dashboard serves as a critical operational and decision-making tool for the finance team, providing clear visibility into transaction volumes, revenue performance, and automated financial reconciliation.

##  Key Features & Responsibilities
* **Data Transformation & ETL:** Imported and transformed raw financial datasets using **Power Query** (Applied Steps: filtering, data typing, merging, and cleaning sheets).
* **Financial Reconciliation:** Developed a data model to cross-reference and reconcile financial inflows and outflows, ensuring data integrity and highlighting discrepancies.
* **Advanced DAX Modeling:** Built complex DAX measures, calculated columns, and time-intelligence functions to analyze trends and Month-over-Month (MoM) metrics.
* **UI/UX Dashboard Design:** Designed a clean, interactive, and user-friendly multi-page dashboard tailored for executive reporting.

##  Tech Stack & Tools
* **ETL & Transformation:** Power Query
* **Business Intelligence:** Microsoft Power BI
* **Analytics Language:** DAX (Data Analysis Expressions)
